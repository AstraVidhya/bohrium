=====================
Fixing Invalid author
=====================

Troels Blum<troels@blum.dk> <Troels Blum<troels@blum.dk>>

This must be done on all branches, othervise merge-conflicts will occur.

On master:

git filter-branch --commit-filter '
        if [ "$GIT_COMMITTER_NAME" = "Troels Blum<troels@blum.dk> <Troels Blum<troels@blum.dk>>" ];
        then
                GIT_COMMITTER_NAME="Troels Blum";
                GIT_AUTHOR_NAME="Troels Blum";
                GIT_COMMITTER_EMAIL="troels@blum.dk";
                GIT_AUTHOR_EMAIL="troels@blum.dk";
                git commit-tree "$@";
        else
                git commit-tree "$@";
        fi' HEAD

On all other branches:

git filter-branch -f --commit-filter '
        if [ "$GIT_COMMITTER_NAME" = "Troels Blum<troels@blum.dk> <Troels Blum<troels@blum.dk>>" ];
        then
                GIT_COMMITTER_NAME="Troels Blum";
                GIT_AUTHOR_NAME="Troels Blum";
                GIT_COMMITTER_EMAIL="troels@blum.dk";
                GIT_AUTHOR_EMAIL="troels@blum.dk";
                git commit-tree "$@";
        else
                git commit-tree "$@";
        fi' HEAD
